01. Smart Fire Alarm System

Fig 1.0 - Rangkaian Skematik Sistem Latching Alarm

Signal Detection (LM393)

INPUT: Sistem memantau suhu lewat NTC Thermistor yang dirangkai sebagai pembagi tegangan bersama Potensiometer.
PROCESS: IC LM393 bekerja membandingkan tegangan referensi (set point) dengan tegangan aktual dari sensor suhu.
TRIGGER: Begitu suhu naik, resistansi NTC bakal anjlok (drop), bikin output LM393 berubah jadi sinyal LOW sebagai tanda bahaya.

The Memory Latch (NE555)

SET: Sinyal LOW dari detektor masuk ke Pin 2 (TRIG) pada IC NE555 untuk mengaktifkan sistem.
LATCH: NE555 bertindak sebagai memori; mengunci Pin 3 (Output) di posisi HIGH untuk menyalakan Buzzer secara konstan.
HOLD: Berkat mode Self-Holding, alarm akan tetap berbunyi walaupun suhu sensor sudah mendingin kembali.

Manual Intervention

RESET: Untuk menghentikan alarm, operator harus menekan Push Button (SW1) guna menarik Pin 4 ke Ground dan melakukan Memory Clear.

Fig 1.1 - Skema Wiring PCB Sistem Latching Alarm

Input Stage (LM393 Connection)

V-REF: Pin 2 LM393 terhubung ke Center Tap Potensiometer (R1).
V-SENSE: Pin 3 LM393 terhubung ke pembagi tegangan NTC (U2) & R2.
OUTPUT: Pin 1 LM393 mengirim sinyal ke Pin 2 (TRIG) IC 555 via R3 (Pull-up).

Control Stage (NE555 Connection)

TRIGGER: Pin 2 nerima LOW dari komparator untuk "ngunci" state.
RESET: Pin 4 dijaga tetap HIGH, ditarik ke GND oleh SW1 untuk reset.
OUTPUT: Pin 3 menggerakkan Base Transistor Q1 (BC547) via R4.

Output & Feedback

ACTUATOR: Buzzer1 aktif saat Q1 saturasi (ON) & volume diatur R5.
FILTER: C1 (100uF) sebagai penstabil tegangan dari VCC ke GND.

Fig 1.2 - Tampilan 3D PCB Sistem

Physical Architecture

FORM FACTOR: Compact Rectangular Design dengan Single-Layer Routing.
COMPONENT: Hybrid System (SMD Resistors + Through-Hole ICs & Actuators).
TRACE LOGIC: Jalur Power (VCC/GND) dibuat lebih tebal untuk stabilitas arus.

Component Placement (UX/UI)

ADJUSTER: Potensiometer (R1 & R5) ditempatkan di pinggir agar mudah di-tuning.
FEEDBACK: Buzzer diletakkan di ujung kiri untuk output suara maksimal.
USER INPUT: Tactile Switch (SW1) di posisi tengah bawah sebagai tombol Reset.

Fig 1.3 - Tampilan Atas PCB Sistem

Top Layer Visualization

SILK-SCREEN: Labeling komponen (U1, Q1, R1, dsb) sangat jelas, memudahkan proses soldering.
ORIENTATION: Notch pada IC U1 & U3 menghadap ke kiri, standar industri untuk mencegah salah pasang.
PAD DESIGN: Menggunakan Gold-plated pads (ENIG style) yang punya konduktivitas tinggi.

Trace & Connectivity Logic

BLUE TRACES: Jalur sinyal di Top Layer didesain rapi tanpa banyak crossing (via-less).
ISOLATION: Jarak antar trace (clearance) cukup aman untuk mencegah short-circuit.
POWER RAIL: VCC dan GND terdistribusi merata ke seluruh blok sensor dan logic.

Fig 1.4 - Tampilan Bawah PCB Sistem

Bottom Layer Visualization

STRATEGY: Single-sided routing (Bottom only) untuk memudahkan proses manufaktur mandiri.
TRACE WIDTH: Jalur distribusi Power & Ground dibuat konsisten untuk meminimalisir resistansi jalur.
SOLDER PADS: Pad komponen Through-Hole memiliki clearance yang cukup untuk mencegah "solder bridge".

Fabrication Details

ANGLE LOGIC: Penggunaan sudut 45 derajat pada belokan jalur untuk aliran sinyal yang lebih baik.
GROUNDING: Jalur Ground terkoneksi secara bintang (Star Grounding) ke pin utama IC.

Phase 1: Thermal Detection

Proses dimulai ketika suhu di sekitar NTC Thermistor (U2) mengalami peningkatan, yang mengakibatkan nilai resistansinya menurun secara drastis. Fenomena ini menyebabkan pergeseran tegangan pada Pin 3 (Non-Inverting Input) IC LM393. Begitu tegangan tersebut melewati ambang batas (Threshold) yang telah dikalibrasi melalui Potensiometer R1, output pada Pin 1 akan seketika berubah menjadi logika LOW sebagai sinyal aktivasi.

Phase 2: Analog Memory Activation

Sinyal LOW tersebut kemudian diteruskan menuju Pin 2 (Trigger) pada IC 555. Di tahap inilah fungsi memori bekerja; IC 555 akan mengubah status Pin 3 (Output) menjadi HIGH dan menguncinya secara permanen (Latch). Dampaknya, meskipun suhu NTC kembali normal dan output LM393 kembali ke posisi HIGH, IC 555 akan tetap mempertahankan suplai arus ke Transistor Q1, sehingga alarm (Buzzer) akan terus berbunyi tanpa henti.

Phase 3: Manual Override

Untuk menghentikan alarm, diperlukan intervensi manual dari operator. Dengan menekan tombol Push Button (SW1), arus pada Pin 4 (Reset) IC 555 akan ditarik menuju Ground (0V). Tindakan ini memaksa sistem untuk menghapus memori pengunci, mengembalikan Pin 3 ke posisi LOW, dan secara otomatis menghentikan aktivitas Buzzer hingga sistem mendeteksi adanya kenaikan suhu kembali.
